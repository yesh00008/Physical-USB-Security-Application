Project Information

This project was developed by Yashwanth , Snehith , Balu as part of a Cyber Security Internship. This project is designed to Secure the Organizations in Real World from Cyber Frauds performed by Hackers.

Project Details

Project Name Physical Usb Security
Project Description This project aims to enhance organizational security by establishing a comprehensive policy governing USB port usage. A Python-based application will enable and disable USB ports, ensuring only authorized devices can connect. Employee training and awareness initiatives will promote understanding of USB security risks. Regular monitoring and audits will be conducted to ensure compliance with the policy. Ultimately, this initiative seeks to mitigate data breaches and unauthorized access through effective control of USB devices.
